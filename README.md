## 🇷🇺 Описание (RU)

Internal DLL для Counter-Strike 2 с overlay-отрисовкой. Инжектится в процесс cs2.exe, читает память игры напрямую через оффсеты, рисует overlay поверх игры через отдельное прозрачное окно (D3D11 + ImGui).

### Особенности
- Все оффсеты берутся из папки `output/` (генерируется cs2-dumper)
- После обновления CS2 достаточно обновить папку `output/` — пересборка DLL не нужна
- При глобальных изменениях движка (например Animgraph 2) может потребоваться правка кода
- Cyberpunk 2077 стиль меню (ImGui)

### Функции

**Combat:**
- Aimbot (FOV до 200, smooth, выбор кости, автовыстрел, velocity prediction)
- Triggerbot (только голова, только с прицелом, проверка видимости)
- RCS (Recoil Control System, регулируемая сила)
- Auto-Stop (остановка перед выстрелом)
- Auto-Knife (автоудар ножом)
- No Recoil / No Spread
- Jump Scout (автовыстрел при приземлении SSG/AWP)

**ESP / Визуал:**
- Box ESP, Bone ESP (MST-алгоритм)
- Имя / Оружие / HP бар / Дистанция
- Off-screen стрелки, Snaplines
- Bomb ESP с таймером
- Damage indicators, Hit sound (регулируемая громкость), Hitmarker (5 стилей)
- Spectator list, Scope sight

**Movement:**
- Bunny Hop, Auto Strafe
- Jump-Duck (прыжок + присед по 1 кнопке)

**Player:**
- Anti-Flash, FOV changer, No Smoke, Radar Hack

### ⚠️ Известные проблемы
- **Skin Changer** — нестабильный, может работать некорректно или крашить
- **Bone ESP** — использует MST-алгоритм вместо фиксированных индексов костей. После перехода CS2 на Animgraph 2 (апрель 2026) скелет может отображаться неточно на некоторых моделях агентов
- **Jump-Duck** — работает нестабильно, может не срабатывать в некоторых ситуациях

### 📌 Статус проекта

Проект больше не планируется активно обновлять. Возможно выйдут обновления, но это не гарантировано. Исходный код открыт — можете форкнуть и дорабатывать самостоятельно.

### Обновление оффсетов

Если вышло обновление CS2 и чит перестал работать:

1. Запусти [cs2-dumper](https://github.com/a2x/cs2-dumper) пока CS2 запущена
2. Скопируй сгенерированную папку `output/` (файлы `offsets.hpp`, `client_dll.hpp`, `buttons.hpp`) туда где лежит DLL
3. Пересобери проект (если у тебя исходники) или просто замени `output/` рядом с DLL

**Важно:** при глобальных изменениях движка (новые системы анимации, изменение структур памяти) простая замена оффсетов может не помочь — потребуется правка исходного кода.

### Сборка

3. Конфигурация: `Release | x64`
4. Build → получаешь `OXRANALOUATABACLIENT.dll`

### Использование

- Инжектить в `cs2.exe` после полной загрузки (главное меню видно)

## 🇬🇧 Description (EN)

Internal DLL for Counter-Strike 2 with overlay rendering. Injected into cs2.exe process, reads game memory directly through offsets, renders overlay via a separate transparent window (D3D11 + ImGui).

### Key Points
- All offsets come from `output/` folder (generated by cs2-dumper)
- After CS2 updates: just replace `output/` folder — no rebuild needed
- For major engine changes (e.g. Animgraph 2 transition): code modifications may be required
- Cyberpunk 2077 inspired premium UI theme

### Features

**Combat:** Aimbot, Triggerbot, RCS, Auto-Stop, Auto-Knife, No Recoil, Jump Scout  
**ESP:** Box, Bones (MST), Name, Weapon, HP bar, Distance, Off-screen arrows, Bomb timer, Damage indicators, Hit sound, Hitmarker  
**Movement:** Bunny Hop, Auto Strafe, Jump-Duck  
**Player:** Anti-Flash, FOV changer, No Smoke, Radar Hack

### ⚠️ Known Issues
- **Skin Changer** — unstable, may not work correctly or cause crashes
- **Bone ESP** — uses MST algorithm instead of hardcoded bone indices. May display inaccurately on some agent models after Animgraph 2 update
- **Jump-Duck** — unstable, may not trigger in some situations

### 📌 Project Status

This project is no longer actively maintained. Updates may come but are not guaranteed. Source code is open — feel free to fork and develop further.

### Updating Offsets

1. Run [cs2-dumper](https://github.com/a2x/cs2-dumper) while CS2 is running
2. Copy generated `output/` folder next to the DLL
3. Rebuild if you have source code

### Building

- Visual Studio 2022, C++17, Release x64
- Open `.sln` → Build
- Output: `OXRANALOUATABACLIENT.dll`

### Controls

- **RShift** — toggle menu
- **END** — unload DLL
